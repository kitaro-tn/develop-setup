# Mac OSX setting script

これはMac book airからproに移行した際に用いたスクリプトである。

## Useage

```
./setup.sh
```

## Scripts

- brew.sh

brewでインストール出来る物をスクリプトに固める

- setup.sh

Macで設定する物をスクリプトにしていく

- Brewfile
